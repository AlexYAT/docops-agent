## Title
Implement Document Creation Invariants

---

## Summary

Реализация базовых инвариантов процесса создания документов
для стабилизации структуры репозитория.

---

## Goal

Сделать процесс создания документов детерминированным
и независимым от ручных решений пользователя.

---

## Scope

### Входит

- фиксированное соответствие doc_type → folder
- генерация ID через Cursor
- стандартизация naming
- формализация workflow создания

### Не входит

- изменения validator
- Document Gate
- автоматизация
- рефакторинг существующих документов

---

## Tasks

### T01 — Define doc_type → folder mapping
- определить фиксированное соответствие типов документов и папок
- зафиксировать правило без исключений

---

### T02 — Define ID generation rule
- описать генерацию ID через Cursor
- запретить ручное задание ID

---

### T03 — Define naming convention
- определить формат имени файла
- связать с ID и title

---

### T04 — Define creation workflow
- описать шаги создания документа
- зафиксировать последовательность действий

---

### T05 — Validate consistency
- проверить согласованность всех правил
- убедиться, что они не противоречат create-only

---

## Done Definition

- определены все правила
- правила согласованы между собой
- можно создать документ без принятия решений пользователем

---

## Notes

- не усложнять
- не добавлять автоматизацию
- фокус на стабильности
