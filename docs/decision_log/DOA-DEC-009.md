# Decision: DOA-DEC-009

## Project
DocOps Agent (DOA)

## Date
2026-03-31

## Status
accepted

---

## Title
Ввести intent-driven auto document generation через Cursor

---

## Context

Ручное создание документов:

- требует выбора шаблона
- требует копирования структуры
- замедляет работу

---

## Problem

Пользователь думает в терминах:

- "надо зафиксировать"
- "надо добавить задачу"

А не:

- "создать decision_log"

---

## Chosen Solution

Перейти к модели:

→ intention → document

через Cursor

---

## Why

- снижает порог входа
- ускоряет работу
- убирает ручные действия
- соответствует реальному мышлению

---

## Trade-offs

- возможны ошибки классификации
- требуется дисциплина prompt

---

## Rule

Пользователь пишет намерение, не документ
