# Decision: DOA-DEC-024

## Project
DocOps Agent (DOA)

## Date
2026-03-31

## Status
accepted

---

## Title
Инициализация проекта с чистой структурой и документацией как источником истины

---

## Context

Проект находится в начальном состоянии:

- отсутствует Git
- документы частично лежат в корне
- структура не организована

---

## Chosen solution

Инициализировать проект:

1. очистить корень (оставить .cursor)
2. создать структуру /docs
3. перенести все документы в соответствующие папки
4. инициализировать Git
5. зафиксировать как baseline проекта

---

## Rule

Документация хранится в /docs и является источником истины проекта
