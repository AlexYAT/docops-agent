# Decision: DOA-DEC-007

## Project
DocOps Agent (DOA)

## Date
2026-03-31

## Status
accepted

---

## Title
Использовать page_id вместо имени страницы для работы с Notion через MCP

---

## Context

При первом использовании MCP:

- поиск страницы "Decisions" по имени не сработал
- была создана новая страница
- возник риск дублирования структуры

---

## Problem

Поиск по имени страницы в Notion:

- ненадёжен
- зависит от контекста workspace
- не гарантирует уникальность

---

## Chosen Solution

Использовать:

→ page_id (или URL страницы)

для всех операций MCP

---

## Why

- точная адресация
- отсутствие дублей
- предсказуемое поведение

---

## Trade-offs

- нужно один раз сохранить ссылки
- чуть менее удобно вручную

---

## Rule

Любая операция MCP должна использовать:

→ URL или page_id родительской страницы
