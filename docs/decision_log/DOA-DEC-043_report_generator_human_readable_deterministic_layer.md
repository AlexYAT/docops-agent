## Title

Добавление человеко-читаемого deterministic слоя в Report Generator MVP

## Context

Текущий Report Generator (T01–T07):

- полностью deterministic
- соответствует anti-SoT
- но недостаточно удобен для восприятия человеком

Отсутствует:

- отображение изменений между запусками
- явная фиксация выполненных действий
- локализация (RU)

## Decision

Добавить в Report Generator:

1. Локализацию заголовков (RU)
2. Новый раздел: «зафиксировано в текущем запуске»
3. Новый раздел: «что изменилось» (delta между запусками)

Все изменения:

- строго deterministic
- без интерпретации
- без NLP
- без рекомендаций

## Constraints

- Report остаётся **НЕ** Source of Truth
- Все строки содержат `(source: ...)`
- Никакой аналитики или выводов
- Только фактические изменения

## Rejected approaches

- Добавление AI-анализа
- Генерация рекомендаций
- Интерпретация данных

## Consequences

**Плюсы:**

- отчёт становится читаемым
- появляется ощущение прогресса
- не нарушается архитектура

**Минусы:**

- усложнение генератора
- появляется зависимость от предыдущего отчёта

## Follow-ups

- обновить operational_plan
- реализовать изменения в generator
- провести повторный audit_check
