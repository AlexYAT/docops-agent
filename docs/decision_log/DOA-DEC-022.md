# Decision: DOA-DEC-022

## Project
DocOps Agent (DOA)

## Date
2026-03-31

## Status
accepted

---

## Title
Актуализация документации через централизованную передачу файлов в ChatGPT

---

## Context

Cursor не имеет гарантированного доступа ко всем ранее созданным документам проекта (decision_log, snapshots, plans).

Это приводит к невозможности корректной актуализации без явного источника истины.

---

## Problem

Попытка "актуализировать документацию" без полного контекста приводит к:

- потере информации
- неполным решениям
- нарушению принципа traceability

---

## Chosen solution

Использовать следующий workflow:

1. Пользователь передаёт актуальные файлы в ChatGPT
2. ChatGPT:
   - анализирует
   - нормализует статусы
   - выявляет новые решения
3. ChatGPT создаёт новые документы (create-only):
   - implementation_snapshot
   - decision_log (при необходимости)
   - operational_plan (новая версия)
4. Cursor используется только как инструмент доставки в Notion через MCP

---

## Why

- единый источник истины
- полный контроль контекста
- предсказуемость системы

---

## Rule

Актуализация документации выполняется только на основе явно переданных документов
