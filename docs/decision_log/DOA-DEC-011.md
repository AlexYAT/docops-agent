# Decision: DOA-DEC-011

## Project
DocOps Agent (DOA)

## Date
2026-03-31

## Status
accepted

---

## Title
Использовать intent-only команды для создания документов

---

## Context

После внедрения auto-doc rules в Cursor появляется возможность
создавать документы без явного указания шаблона.

---

## Chosen Solution

Пользователь пишет только намерение:

→ "зафиксируй состояние системы"

Cursor:

→ определяет тип документа  
→ генерирует по шаблону  
→ отправляет через MCP  

---

## Why

- минимизация усилий
- соответствие мышлению пользователя
- ускорение работы

---

## Trade-offs

- возможны ошибки классификации
- требуется корректно настроенный system prompt

---

## Rule

Пользователь не указывает тип документа вручную
