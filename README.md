# DocOps Agent

Проект: **DocOps Agent** — docs-first система управления разработкой.

## Основные принципы

- **docs-first** — документация и соглашения первичны относительно кода.
- **planning-first** — планирование и явные планы задают рамку работ.
- **create-only** — содержательные изменения вносятся созданием новых артефактов, а не правкой истории.
- **explicit decisions** — значимые решения оформляются и фиксируются явно.
- **traceability** — связь между этапами, решениями и снимками состояния прослеживаема.
